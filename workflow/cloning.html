Cloning

Why use cloning?

Often cloning is used when first setting up a repository for a new project. After forking the repository from its original creator, to your own github profile. You may then CLONE the repository from githubs servers to your own machine. You often edit changes on your own machine then push the repository back up to the 'origin' (in this case the github server).

Cloning is used to...

Set up a repository from a origin repo.

How do you use cloning?
