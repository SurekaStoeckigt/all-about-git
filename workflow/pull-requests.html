Pull Requests

Why use pull requests?

Pull requests are used to synchronise your local repository with the remote one by taking (‘pulling’) changes from the remote repository and applying them immediately to the local repository. It combines the ‘fetch’ and ‘merge’ requests into one process.

How do you use pull requests?

git pull <remotegt; <local>
