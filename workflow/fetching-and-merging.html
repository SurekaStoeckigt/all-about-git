Fetching & Merging

Why use fetching and merging?

Fetching and merging is used to update your local repository whenever a remote repository has been changed.

How do you use fetching and merging?

                
    
                    
  • Let's say there is a remote repository called 'upstream'.
    • 
                    
  • To fetch it, type 'git fetch upstream' in your command line.
    • 
                    
  • To then merge it with the local repository, type 'git fetch upstream/master' to make sure the master branch of your local repository is in sync with the uspream repository.
